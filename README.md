# Dr. Chandrashekar Clinic Website

A professional, responsive website for Dr. Chandrashekar, MD, DM (Cardiology) — an Interventional Cardiologist with 26,000+ procedures and 26+ years of experience.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile
- **Hero Section** - Eye-catching banner with key credentials and call-to-action buttons
- **About Section** - Doctor profile with animated statistics
- **Education Section** - Academic qualifications and credentials
- **Services Section** - Comprehensive cardiac services offered
- **Talk to Us** - Dedicated contact section with phone and WhatsApp
- **Contact Section** - Full clinic address, phone, WhatsApp, and embedded map
- **Floating Action Buttons** - Quick access to call and WhatsApp from any page
- **Smooth Animations** - Scroll-triggered animations and counter effects

## Quick Deploy to Vercel

### Option 1: Deploy via Vercel CLI

```bash
# Install Vercel CLI
npm i -g vercel

# Navigate to project folder
cd dr-chandrashekar-clinic

# Deploy
vercel
```

### Option 2: Deploy via GitHub + Vercel

1. Create a new GitHub repository
2. Upload all files from this folder to the repository
3. Go to [vercel.com](https://vercel.com) and sign in with GitHub
4. Click "Add New Project" and select your repository
5. Vercel will auto-detect the static site and deploy it
6. Your site will be live at `https://your-repo-name.vercel.app`

### Option 3: Deploy via Vercel Dashboard (Drag & Drop)

1. Zip all files in this folder
2. Go to [vercel.com](https://vercel.com)
3. Click "Add New..." → "Project"
4. Choose "Import Git Repository" or drag & drop your zip
